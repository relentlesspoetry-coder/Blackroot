## A detour worth recording

I spent **three rounds** convinced the prompt had a stale-reference bug â€” inspecting DOM nodes, checking
`isConnected`, comparing cached vs queried elements, hunting a detached panel.

**There was no bug.** My test character had **died**: I'd teleported him to `+40,+40` in the wilderness
in an earlier step and left him there. `refreshExaminePanel` correctly hides for a dead player, so the

All that reasoning found nothing because there was nothing to find. **One line of instrumentation
printed `playerAlive: false`** and it was over. The lesson is the same one this whole session keeps
teaching from the other direction: measure the thing, don't reason about it.
