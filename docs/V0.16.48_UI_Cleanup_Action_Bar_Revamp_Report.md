# V0.16.48 - UI Cleanup and Action Bar Revamp Report

## Summary
Implemented the requested UI cleanup phase from the current V0.16.47 baseline, advancing the project to V0.16.48. Scope was limited to the Character Stats window fit pass, bag item slot info presentation, and action bar visual revamp.

## Root causes found
- The Character Stats mockup layout used presentation-scale paper-doll constants that were too aggressive for common desktop heights, causing equipped class models and props to clip in the ornate viewport.
- The Character Stats window had oversized header, panel padding, slot spacing, and stat-card spacing for a viewport-constrained desktop layout, which could push content behind the frame.
- Bag item slots were rendering only icon, name, rarity, and class restriction on-card; key stats were only available in hover tooltip text.
- The action bar still used compact slot geometry and small icon sizing that did not match the new premium Blackroot UI direction.

## Character Stats layout fixes
- Reduced oversized header/padding/card spacing in the existing ornate Character Stats layout so content fits more reliably inside the window.
- Kept the same two-column paper-doll/stat structure and preserved all displayed equipment/stat/set bonus information.

## Model viewport fixes
- Rebalanced `renderPaperDollClassModel()` scale clamps and foot-anchor placement to fit tall class props, weapons, and robe silhouettes inside the model viewport.
- Preserved the canonical class renderer and equipment rendering path; no duplicate preview system was added.

## Bag item card changes
- Added `bagItemSlotSummaryLines()` in the inventory system to derive compact on-card stat summaries from the same normalized item stat data used elsewhere.
- Bag cards now show direct stat lines where space allows while retaining full hover tooltips and existing click/right-click/double-click behavior.

## Action bar renderer changes
- Reworked the existing `#hotbar.abilityBar` CSS into a larger ornate dark-fantasy frame with an integrated XP meter, larger icon-first slots, gold trim, purple glow states, and a centered Meditate button.
- Preserved slot order, keybinds, spell usage, cooldown readouts, Auto Attack, and hover/click routing.
