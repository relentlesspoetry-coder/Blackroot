# V0.20.28 — The Wizard's Elements Finally Look Different (Roadmap Item 6)

## The bespoke pass begins — and the Wizard's gap is the sharpest

The Wizard is the class whose whole identity is **elemental force**, and every one of its spells drew
**the same wizard-blue rune ring**. A Fireball, a Frost Nova and an Arcane Bolt were visually
identical.

Palette identity (V0.20.21) made the Wizard look distinct from a *Necromancer* — but did nothing
*within* the Wizard, because fire, frost and arcane all resolve to `sourceClass=wizard → rune/blue`.
The class palette is the wrong tool for intra-class element variety.

## The fix — an isolated bespoke element burst

`drawWizardElementEffect` is a **new, isolated effect type** (`wizardElement`) that nothing else
spawns, so it differentiates the elements **without touching the shared `vfxStyle` palette** the other
13 classes depend on:

| element | look |
|---|---|
| **fire** | orange bloom + rising embers |
| **frost** | cyan ring throwing ice shards |
| **lightning** | yellow crackling arcs |
| **arcane** | violet rune geometry |

**Element is derived from `damageType` + `tags` + `id`, never from `buff.color`** — Ignite ships the
wizard's blue, the same lie the DoT plume caught in V0.20.25. Keying off colour would make fire read as
frost.

Wired into the Wizard's **10 damage spells**: AoE bursts at the center (keeping V0.20.23's accurate
area ring underneath), single-target bursts at the target (keeping the travelling bolt).

## Scope

This is the Wizard's damage-spell elemental identity. It sits **on top of** the existing fidelity work
(accurate AoE rings, travelling bolts at hand height, DoT plumes) rather than replacing it. It does not
touch the Wizard's utility/buff spells (Blink, Overchannel, etc. — those keep their rings), and it
leaves every other class's VFX untouched by design.

## The lesson, reused

The DoT-plume finding, applied again: **the element lives in `damageType`, not `buff.color`.** Every
time this session has needed an element, the authored colour has been wrong (Ignite is blue), and the
damage type has been right. Bespoke art is only as honest as the field it keys off.
