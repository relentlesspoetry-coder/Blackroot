# V0.16.74 - Status Timer Labels Report

## Scope
Adds visible remaining-time labels below the existing circular buff/debuff/status icons.

## Changed Files
- `systems/status-effect-system.js`
- `Blackroot V0.16.74.html`
- `core/config.js`
- `VERSION.txt`
- `README.txt`
- `PATCH_NOTES.md`

## Behavior
- Each status icon keeps the radial conic countdown from V0.16.73.
- Each visible icon now displays a compact timer label directly below the circle.
- Labels are generated from `effect.remaining`, matching the same value used by the circular mask.
- Short durations display as seconds. Longer durations display as `m:ss`; hour-long statuses display as `h:mmh`.

## Non-Goals
- No spell balance changes.
- No mob/debuff data changes.
- No inventory, save, combat, or action-bar assignment changes.
