- Spell names
- Unlock levels
- Mana costs
- Cooldowns
- Damage/heal/effect values
- Targeting/ranges
- Action-bar mappings
- Tooltip text
- Save/load spell assignment behavior

Existing Assassin characters should pick up the icons automatically because spell icons resolve from the live compiled spell definitions by class/name.
