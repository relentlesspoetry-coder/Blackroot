# V0.16.66 - Action Bar Layout Refinement Pass

## Summary
Refined only the bottom action bar layout. Spell icons now use more of the slot interior, the EXP meter uses a purple fill/dark-purple empty gradient, buffs and debuffs are split around the Meditate button, and the Meditate button is mounted on the action-bar top edge instead of intruding into the bar contents.

## Spell Icon Slot Changes
The existing `.actionSlot`, `.spellActionIcon`, and `.hotbarSpellIcon` structure remains the owner. Slot icon cells were enlarged, icon inner padding was removed for hotbar image icons, and image icons now scale slightly inside the same hotbar icon node so the uploaded spell art fills the intended interior without creating a second renderer or changing icon assets.

## EXP Bar Changes
The action-bar EXP meter remains the existing `#xpFill` / `#xpText` owner. Its empty state now uses a dark purple gradient, and the fill uses a stronger purple gradient. EXP values and text formatting remain unchanged as `Exp: current/max`.

## Buff/Debuff Layout Changes
The previous single action status tray displayed all player effects together. The hotbar now owns an `actionStatusRow` with two trays: `actionBuffTray` on the left side of the Meditate button and `actionDebuffTray` on the right. `StatusEffectSystem.renderTray()` now accepts an optional hostile filter so the existing status renderer can be reused without duplicating tray rendering or tooltip logic.

## Meditate Button Changes
The existing `#steamDeckMeditateBtn` remains the only Meditate button. It was moved to the action-bar top edge by adjusting the owning CSS layout so it rests above the content instead of overlapping the EXP meter. Existing hover, pressed, active, disabled, click, and keybind behavior were preserved.

## Preserved Gameplay Behaviors
- Auto Attack remains key `1`.
- Spell slots remain keys `2-0`.
- Slot click/touch/keyboard behavior remains delegated through the existing hotbar input owner.
- Cooldown overlays and resource/disabled states remain active.
- Meditation logic and keybind behavior were not changed.
- Spell behavior, combat behavior, inventory, settings, stash, map, quest log, and dialogue systems were not modified.
