# V0.18.86 — Dark Woods Offhands (Roadmap Item 4 B)

The **offhand** slot gets a full pool of 10 — pairing with the main-hand weapons from V0.18.85. Two
kinds, matching the four pre-existing offhands: **shields** (`type: armor`, add armour + defense/hp)
for the front line, and caster **focuses** (`type: accessory`, `armor: 0`, add mana/spellPower/
healing) for spellcasters. Same rarity spread (2 white / 5 green / 3 blue), audit-guarded to 0
unsourced.
