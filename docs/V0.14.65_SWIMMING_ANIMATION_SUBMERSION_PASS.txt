Dream Realms V0.14.65 — Swimming Animation Submersion Pass

Summary
- Adds swimming and treading animations for every playable class when the player enters water.
- Uses the existing Swimming / Underwater Exploration system and waterDepthAt terrain owner for rivers, ponds, lakes, oceans/future water volumes, and cave pools.
- Keeps the character visually waist-deep through class renderer swim poses and layered waterline submersion/ripple overlays.
- Preserves combat/cast/gather/fishing/meditation/death state priority; swimming only owns locomotion/idle water behavior.
