### How bags work (verified)
Bags compile to `runtimeKind: 'material'` — **exactly like the existing `item_linen_pouch`**. The inventory bag system (`systems/inventory-system.js`) recognises a bag via `isBagItem` (`type === 'bag'` **or** `stats.slots > 0`) and reads capacity from `bagSlots`/`stats.slots`, not from a `'bag'` runtimeKind. Verified on the runtime instances: `isBag: true` and capacity **5..14**, matching the existing bags.
