# V0.20.29 — Every Class Now Has A Face (Roadmap Item 6, caster bespoke pass complete)

## The last five

Druid, Shaman, Summoner, Enchanter and Ranger were the five casters still throwing the same generic
ring for every spell. This gives each a bespoke burst — and completes the class-VFX work.

Built as **one shared isolated effect type** (`casterMotif`) keyed by **motif**, so five classes reuse
one draw path instead of five near-duplicates — the same isolation discipline as the Wizard's
`wizardElement` (V0.20.28): nothing shared with `vfxStyle`, so the eight already-done classes are
untouched.

## The motifs — from each class's own data, never `buff.color`

| class | motif(s) | look |
|---|---|---|
| **Druid** | nature / lunar | green leaves + sprigs / pale-violet moon glow + crescent |
| **Shaman** | storm / earth / spirit | teal crackling arcs / amber radiating shards / drifting wisps |
| **Summoner** | planar | violet vertical rift shimmer |
| **Enchanter** | mind | rose concentric psychic pulses |
| **Ranger** | hunt | moss crosshair + quick streaks |

Element/motif is derived from `school` + `damageType` + `tags` + `id` — **never `buff.color`**, the lie
this session has caught repeatedly (Ignite ships blue).

## Wired at one point per class

Each resolver has a single damage helper, so **one edit per class** covers all its spells:

| class | injection point |
|---|---|
| Shaman | `hit()` |
| Druid | `damageEnemy()` |
| Enchanter | `hitEnemy()` **and** `applyControl()` |
| Summoner | `damage()` |
| Ranger | both damage sites (single-target + AoE loop) |

**The Enchanter fires on control, not just damage** — it's a control class (mesmerize/charm/sleep), so
bursting only on damage would leave most of its casts blank. Verified: Mesmerize spawns a `mind` motif.

## Honest scope

These are burst *motifs* — a coherent per-class visual language at the impact point — not per-*spell*
hand-drawn art. A Druid's Thorn Whip and Moonfire differ (nature vs lunar) but two nature spells share
the nature burst. That's the same bar as the Wizard's elemental bursts, and it's a real, shipped floor:
every class is now visually distinct and reads its own identity. Per-spell bespoke art (the way the
Assassin has a unique look per ability) remains a deeper polish available class by class.

## The lesson, one last time this arc

Eight versions of VFX work (V0.20.21→29), and the same rule held every time: **the identity lives in
what the spell does — its class, its damage type, its school — never in the colour someone typed on it.**
`buff.color` was wrong for fire, wrong for the caster funnel, wrong for the DoT plume. Keying off
behaviour, not decoration, is what made all fourteen classes finally look like themselves.
