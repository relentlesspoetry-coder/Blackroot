# V0.16.86 - High-Quality Procedural VFX Renderer Revamp Report

## Effect types upgraded

### Generic effects
- `ring`
  - Now renders layered ground ellipses, soft glow, expanding ellipse rings, inner pulse, and motes/smoke/runes depending on style.
- `bolt`
  - Now renders a world-space traveling core, tapered ribbon trail, secondary arcs, projectile head, and impact bloom.
- `statusPulse`
  - Now renders layered ground contact, glow bloom, expanding ellipses, motes, smoke, or runes depending on class/school/kind.
- `combatImpact`
  - Now renders contact shadow, impact bloom, shock ring, directional spark burst, and lingering smoke/motes.
- `combatSpark`
  - Now renders short directional spark streaks instead of square pixels.

### Assassin effects
- `assassinProjectile`
  - Upgraded knife, crossbow, and poison dart projectile motion with tapered trails, projectile cores, afterimages, poison motes, and marked-target flare rings.
- `assassinImpact`
  - Upgraded impacts with bloom, shock ring, spark fan, poison globules, and mark sigil flare.
- `assassinCrosshair`
  - Upgraded aim cue with shrinking sigil/crosshair, glow, and smoke residue.
- `assassinTripwire` / `assassinTripwireTrigger`
  - Upgraded line/wire rendering with ground contact, stake anchors, trigger snap, and spark burst.
- `assassinPoisonPulse`
  - Upgraded poison ticks with bubbling motes, sickly glow, ground residue, and smoke wisps.
- `assassinMarkLine`
  - Upgraded mark application line with ribbon trail and bright inner tracer.
- `assassinMarkSigil`
  - Upgraded persistent mark with pulse glow, rotating sigil ring, dark ground shadow, and blood-drop drift.
- `assassinMarkFlare`
  - Upgraded hit flare with bloom, execution sigil expansion, and spark burst.
- `assassinMarkRemove`
  - Upgraded removal with smoke wisps, red shard burst, and dissipating ground shadow.

## Class/school style routing
The renderer now routes effect appearance using `sourceClass`, `school`, `kind`, `style`, and label metadata:

- Poison / venom: green bubbles, sickly motes, smoke residue.
- Assassin: narrow precision trails, steel/black/red accents, lethal mark sigils.
- Necromancer: bone/soul/green smoke/wisp behavior.
- Wizard / arcane / fire / frost / lightning: clean runes, precise cores, star-like fragments.
- Cleric / holy / heal: warm bloom, gold/white motes, restorative upward drift.
- Druid / Warden / nature: organic motes, ground anchoring, leaf/spore-like motion.
- Bard: rhythmic arc/soundwave styling.
- Fighter / slash / impact: steel/orange spark and dust impact language.

## Performance safeguards
- Existing active effect and damage text budgets are preserved.
- Existing object pooling remains intact.
- Renderer-side `effectQualityScale()` reduces spark/mote counts when active effect load rises.
- Core projectiles, impacts, and telegraphs are kept readable before ambient particles are reduced.
- No external assets, WebGL, Three.js, shaders, sprite sheets, or new dependencies were added.

## Protected systems
The following were not intentionally changed:

- spell damage
- spell cooldowns
- target selection
- status logic
- class mechanics
- save/load behavior
- inventory/bags
- map systems
- quest/dialogue systems
- UI layout
- combat math

## Known limitations
- This phase upgrades the VFX renderer and representative generic effects first.
- Many class-specific spell visuals can now be improved faster through the shared toolkit, but not every individual spell has bespoke art direction yet.
- The renderer remains Canvas 2D procedural by design.
