## Confirmed Root Cause
The missing Logout and Exit Game buttons were caused by malformed/stale in-game Menu Patch Notes markup. The large embedded `#menuPatchNotesPanel` contained the patch-notes entries but was missing its own closing container before `#menuLogoutBtn` and `#menuExitBtn`. As a result, Logout and Exit Game were DOM children of the hidden Patch Notes panel instead of the visible `.menuList`, so they disappeared whenever Patch Notes was closed. This also made the Menu look clipped/shortened.

The Player HUD / Party HUD issue was a UI visibility ownership regression: gameplay HUD chrome relied on body-level `gameStarted` gates and normal CSS display defaults, but later HUD/menu layout changes could leave gameplay chrome without an explicit in-game restoration pass. The existing unified Party / Companions HUD also depends on remaining attached inside `#hud`; if the player HUD is hidden or not reasserted, party/companion cards disappear with it.

The cause was UI markup ownership plus HUD visibility restoration, not combat, save/load, loot, class rendering, or cape geometry. No duplicate renderer or duplicate HUD system was required.

## Fixes Implemented
- Corrected the static in-game Menu Patch Notes markup so `#menuPatchNotesPanel` closes before Logout and Exit Game.
- Added `ensureMenuButtonOwnership()` to the existing UI owner. It moves existing Logout/Exit buttons back into `.menuList` if stale markup ever nests them inside Patch Notes, and applies scroll-safe Menu sizing.
- Added `ensureGameplayHudChrome()` to the existing UI owner. It reasserts `body.gameStarted`, visible `#hud`, visible `#menuDock`, and the unified companion HUD attachment during gameplay HUD refreshes.
- Called the restoration hooks from normal gameplay entry, full HUD updates, fast HUD updates, and Menu open handling.
- Updated Menu CSS so the full Menu scrolls cleanly instead of clipping bottom buttons.
- Refreshed embedded Patch Notes panel content from `PATCH_NOTES.md` so the added V0.17.07 entry appears in both Patch Notes panels.
