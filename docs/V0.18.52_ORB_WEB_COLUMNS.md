# V0.18.52 — Silk Web Caverns: real spider-web columns + faster crawlers

Your note: *"the tiny spiders move too slow on the walls and nests, also these webs need to be revamped to look like real spider webs or just removed (image 1)."*

Showcase of the rebuilt columns: [V0.18.52_orb_web_columns.png](V0.18.52_orb_web_columns.png) — thick, thick, medium, small, royal.

## 1. The web columns are rebuilt as real spider webs
The floor-to-ceiling web columns used a woven **net node grid** (rows × columns of nodes joined by vertical + horizontal + diagonal threads). Even after heavy jitter (V0.18.50), that keeps a **grid topology**, so it still read as a rectangular fishing net — which is what you kept seeing.

A real spider web is **radial**, not a grid. So the net is **removed entirely** and `drawCeilingWebColumn` (`render/object-renderer.js`) is rebuilt as a vertical **stack of orb webs**:

- **thick = 3 orbs, medium = 2, small = 1**, spaced up the column and overlapping into one continuous web.
- each orb is a proper orb web — **radial spokes + a dense capture spiral + a frame ring** — via the shared `drawOrbWeb`.
- they hang on an **organic scaffold** instead of a grid: a few slightly-splayed, sagging **spine strands** run floor → each orb hub → ceiling, plus jittered **guy-lines** from each orb's frame out toward the edges.
- `drawOrbWeb`'s capture spiral now **scales with radius** (orbs bigger than ~30px get ~4.6 turns and finer steps), so the large column orbs stay dense rather than looking like a coarse polygon.
- kept: the translucent silk sheet (thick/medium), the ceiling anchor fan, the floor silk pool, clinging egg sacs, the woven web nest (thick), and the base bones (thick).

Result: each column now reads unmistakably as a stack of spider webs (see showcase) — no net grid.

## 2. The crawling spiders are faster
The animated tiny spiders were correct but too slow. Their `performance.now()` time multipliers were raised ~3–4×:
- **Wall spiders** (`render/terrain-renderer.js` `drawSilkCavernWallWebs`): drift-across `0.00022 → 0.00078`, drift-up/down `0.00018 → 0.00064`, turn `0.00025 → 0.0009` (amplitudes nudged up too).
- **Nest spiders** (`render/object-renderer.js` `drawGiantSpiderNest`): orbit speed `~0.00005–0.00011 → ~0.0002–0.00044`, breathing `0.0004 → 0.0013`.
