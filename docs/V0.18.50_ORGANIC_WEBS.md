# V0.18.50 — Silk Web Caverns: webs made organic (less geometric)

You sent an in-game screenshot of the webs and said "continue." I read it as: the floor webs and the web columns **still look too geometric** — a field of bright dots + straight star-spokes on the floor, and a rectangular fishing-net grid on the columns. This version softens both toward organic spider silk. (If you meant something else, tell me.)

Showcase: [contiguous floor + columns](V0.18.50_organic_webs.png) — the floor now reads as soft silk with soft coiled orb webs, and the columns are irregular spun webs with orb webs instead of a rectangular grid.

## What was making it look geometric
1. **Bright junction dots everywhere** — the floor put a bright bead on nearly every web anchor and on a ring of orb-web junctions. Across many tiles those dots read as the vertices of a regular mesh.
2. **Straight radial orb spokes** — at game zoom the orb webs' straight spokes read as repeating star-bursts / triangles.
3. **Rectangular column net** — the ceiling columns used a low-jitter node grid, so vertical + horizontal threads formed rectangular cells (a fishing net, not a spun web).

## The changes (2 files, render-only)
### Floor — `render/terrain-renderer.js` `drawSilkWebCavernSurfaceDetails`
- **Anchor-knot beads**: only a couple, dim (was a bright bead on most knots) — threshold `0.5 → 0.74`, alpha `0.5 → 0.3`, radius `0.8 → 0.6`.
- **Orb junction dots**: the whole ring of junction beads is dropped, leaving just one soft centre bead.
- **Orb spokes**: dimmed + thinned (alpha `0.36 → 0.24`, width `0.7 → 0.55`).
- **Capture spiral**: now the dominant feature — denser (`~2.2 → ~3.0` turns, finer step) and brighter (alpha `0.32 → 0.42`), so the soft coil of a real orb web reads, not the straight spokes.

### Columns — `render/object-renderer.js` `drawCeilingWebColumn`
- **Node-grid jitter** raised so the cells are **irregular** (a spun web) rather than a rectangular grid: x `0.16 → 0.44` of the local width, y `7 → 13` px.
- **Embedded orb webs** raised (thick `2 → 3`, medium `1 → 2`, small `0 → 1`) so the orb webs dominate the column.
- **Node beads** made sparse + dim so they don't re-form a dotted grid.

No layout / config / gameplay changes; no other files touched.
