# Dark Woods 360x360 Expansion — Phase 26: Collision + Readability Cleanup

**Phase name:** Collision + Readability Cleanup (Phase 26 of 28)
**Previous version:** V0.17.64
**New version:** V0.17.65

---

## Goal

Clean up movement/collision/readability after all the landmarks, resources, and props added across
Phases 2–25: audit roads, cave entrances, quest/combat spaces, the waypoint centre, resource nodes,
fishing spots, major landmarks, tree barriers, large-prop collision, and map edges; fix any accidental
blockers, interactables under collision, unreachable nodes, blocked fishing spots, waypoint/cave-access
issues.

## Outcome: AUDIT ONLY — no issues found, no functional changes

This is an audit phase (like Phases 7 and 18). A real reachability sweep confirmed that everything added
in Phases 2–25 is reachable and non-blocking, so **no game-code changes were needed**. The careful,
`isWalkable`-verified placement used in Phases 20–25 (resource nodes) and Phase 23 (dungeon water pools)
already avoided collision problems. Version/docs updated only.

---

## Audit performed (in-browser, deterministic maps)

### Overworld
- Flood-filled walkable tiles from the starter spawn (100,100): **113,621 tiles reachable** — the
  360x360 zone is one large connected walkable area.
- **All 18 herb nodes** (Phase 20) are on walkable tiles **and** reachable from spawn (0 issues).
- **All key landmarks reachable:** waypoint centre (210,250), Silk Web Cavern entrance (52,139),
  Hidden Tree Cave (76,74), Bandit's Fall (~230,65), starter spawn.
- **Map edges blocked:** all 6 sampled edge tiles are non-walkable (the zone is properly bounded).

### Silk Web Cavern (all 3 dungeon floors)
- **All dungeon herb nodes** (Phase 21) reachable from the floor entrance (0 unreachable).
- **All carved fishing pools** (Phase 23) have reachable walkable shore tiles, so every pool is
  fishable (the pool *interior* tiles have no shore, which is expected — you cast from the edge).
- **Zero water tiles in any boss arena** (the deep pool sits at a deep non-boss room, not the arena).
- **Floors stay well-connected** (11,964 / 12,422 / 13,090 reachable tiles on floors 1/2/3), so the
  Phase 23 water pools did **not** disconnect any room or corridor.

### Not changed
- Waypoint centre kept clear since Phases 10–12; cave entrances working since Phases 5–8; tree/rooted
  barriers and ruin walls block only where intended. Non-blocking props (leaves/ferns/etc.) may visually
  sit near nodes but do not affect collision or reachability.
