# V0.16.84 - Necromancer Spell Revamp + Icon Integration Report

## Source files used

- Spell document: `Necromancer Spells(1).txt` / uploaded `Necromancer Spells.txt`.
- Icon package: `Necromancer Spell Icons.zip` / uploaded `Necromancer Spell Icons(1).zip`.

## Spell data owner updated

- `data/spells.js`
  - Replaced the Necromancer Level 1-20 spellbook with the uploaded balance-pass spell order and values.
  - Preserved stable `necroSpellId` keys for save/action-bar compatibility where possible.
  - Necromancer remains Mana-based.

## Behavior owner updated

- `systems/spell-system.js`
  - Updated the existing Necromancer resolver rather than adding a duplicate spell system.
  - Kept spell behavior in `resolveNecromancerClassSpell()` and supporting Necromancer helpers.
  - Updated disease, curse, drain, bone, pet repair, pet summon, temporary undead, Death Pact, Lich Veil, Soul Leech, and Grave Sovereign behavior.
  - Added direct use of Plague Mark and Bone Shackles bonuses in the existing damage/DoT helpers.

## Pet owner updated

- `entities/pet.js`
  - Added support for explicit main-pet lifetime and temporary-pet attack limits through existing pet entity fields.
  - This supports Raise Skeleton / Summon Bone Servant 1-hour duration and Army of Bones attack limits without changing pet AI, commands, pathing, stats, or visuals.

## Icon assets added

- `assets/spell-icons/necromancer/`
  - Added 20 PNGs using canonical snake_case names.
  - Preserved uploaded `manifest.txt`.
  - Added generated `manifest.csv` documenting spell-to-icon mapping.

## Icon registry updated

- `systems/spell-system.js`
  - Replaced old Necromancer icon aliases with direct V0.16.84 mappings.
  - Retained compatibility aliases for older Necromancer draft IDs such as `bone_spear`, `grave_armor`, `raise_bone_servant`, `summon_skeleton_knight`, `grave_nova`, and `lich_form`.

## Manifest and mapping

- The uploaded `manifest.txt` was preserved.
- All new icons mapped directly by spell name and level.
- No Necromancer icon mismatches were found.

## Fully implemented mechanics

- Mana-based Necromancer spellbook order and costs.
- One-main-undead replacement rule for Raise Skeleton and Summon Bone Servant.
- Default undead pet command set remains Assist through existing pet command owner.
- Bone Splinter diseased-target bonus.
- Grave Rot disease DoT refresh path.
- Life Tap self-only drain sustain.
- Bone Armor charged physical mitigation.
- Plague Mark single-target replacement and disease/rot amplification.
- Corpse Mender undead-minion-only HoT with healing cap metadata.
- Rot Cloud disease AoE DoT.
- Bone Shackles root plus bone-damage setup.
- Withering Curse DoT plus enemy damage-reduction debuff.
- Army of Bones temporary skeleton count, duration, and attack-limit support.
- Death Pact pet-health sacrifice gate and 10-spell damage window.
- Bone Storm caster/minion-centered AoE DoT path.
- Lich Veil drain sustain / shadow-resistance buff.
- Soul Leech channel-style drain DoT and self-heal path.
- Grave Sovereign pet champion buff, pet HP scaling, pet damage/attack-speed buff, and caster necromancy boost.
