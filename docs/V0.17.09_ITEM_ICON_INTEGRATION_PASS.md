## Implementation

- The exporter crops only each 120×120 framed artwork cell, excluding all source-sheet label text.
- Runtime icon filenames use canonical item IDs without the `item_` prefix.
- Canonical item records receive `iconKey` values from a stable ID list in `data/items.js`.
- Older save objects without `iconKey` resolve through `DR.ITEM_BY_ID` using their stable item ID.
- `Game.prototype.itemIconDescriptor` remains the single lookup path and now returns an `imagePath` when art exists.
- `Game.prototype.itemIconHtml` renders image-backed art for mapped items and preserves procedural glyph fallback for unmapped items.
