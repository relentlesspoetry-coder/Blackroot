# V0.16.56 - Main Menu Floating Class Icon Safe-Zone Pass

## Summary
Converted the main-menu floating class sigils from fixed CSS-only drift paths into a deterministic safe-zone-aware movement pass owned by `systems/ui-system.js`. The pass keeps icons animated while preventing overlap with the painted logo, menu-button stack, active modal panels, and other class icons.

## Root cause found
The prior floating icon implementation used static viewport percentages plus CSS keyframe offsets. Because the logo is painted by `#logoSplash::before` and the menu stack has independent responsive bounds, the CSS paths had no knowledge of protected UI regions. Bard could drift into the logo region, Enchanter could drift into the menu-button column, and icons could overlap one another because no collision radius was enforced.

## Protected zone implementation details
- Added `buildMainMenuProtectedZones()` in the UI owner.
- The logo zone is viewport-relative because the logo is rendered by `#logoSplash::before`, not a queryable image element.
- The menu-button zone is derived from the live `.splashActionRow` DOM bounds with padding.
- Active modal panels are included as protected zones when open.

## Icon collision/spacing implementation details
- Added per-icon deterministic movement state keyed by class name.
- Each icon gets a radius derived from its emblem size plus glow padding.
- Each update resolves circle-vs-rectangle overlap against safe zones.
- Pairwise icon spacing prevents class icons from touching or overlapping.
- Movement remains smooth through velocity reflection and small deterministic drift acceleration; no respawning, hiding, setTimeout repair, or duplicate icon system was added.

## Viewport/resolution behavior notes
- Safe zones are recalculated from the current splash viewport and live menu-button bounds.
- Icon positions are clamped to the visible screen with radius padding.
- Resizing rebuilds deterministic icon state using the existing class specs and immediately resolves it against the protected zones.
