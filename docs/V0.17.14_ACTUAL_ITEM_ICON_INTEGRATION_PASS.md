## Root Cause and Implementation

- The project already had canonical icon keys for 153 items and one shared renderer used by inventory, bags, equipment, Character slots, loot, shops, rewards, tooltips, bank, trade, mercenary, and bot surfaces.
- Previous runtime files were 120×120 crops exported from labeled sheets. Comparison with the new ZIP showed those crops cut into complete outer frames.
- `Item Icons.zip` contains 153 separate finished 128×128 RGBA icons. These are now copied byte-for-byte into the existing canonical runtime filenames.
- The stale sheet-crop exporter was replaced by `tools/import-item-icon-pack.py`, preventing future rebuilds from restoring the clipped art.

## Matching and Audit

- 153 icons successfully mapped.

- 61 canonical game items have no icon in the uploaded pack and retain their existing generated fallback.
