# V0.20.30 — Bosses Fight Back, Tooltips Tell The Truth

Two pending decisions, resolved with the user's chosen values.

## 1. Boss CC resistance (Roadmap Item 19)

The V0.20.27 gate shipped but **no enemy used it** — the lever was installed, unpulled. The user picked
the **Moderate** tier scheme, wired at the one funnel (`baseControlResistance`) rather than editing 28+
enemy data entries:

| tier | resist | a 3.2s stun becomes |
|---|---|---|
| `dungeonBoss` | 0.55 | **1.44s** |
| `dungeonMiniBoss` | 0.40 | **1.92s** |
| `elite: true` | 0.20 | **2.56s** |
| normal | 0.00 | **3.2s** (unchanged) |

**An explicit `controlResistancePct` on the entity/baseType still wins**, so any single enemy stays
hand-tunable. **Players carry none of these flags**, so their base stays 0 and only their buffs
contribute — V0.20.27's player-side behaviour is untouched.

| **player** | 3.2 | ✅ (unaffected) |
| player + 0.5 buff | 1.6 | ✅ (V0.20.27 preserved) |

## 2. Channel tooltips (Roadmap Item 6)

Seven spells claimed to "channel for N seconds" but resolve **instantly** — the engine has no channel
state (verified in the earlier audit). The user chose **reword to match**. All 7 now describe the real
instant multi-hit:

| spell | before → after |
|---|---|
| Whirlwind Cleave | "Channel a spinning attack for 3 seconds, over 3 ticks" → "Spin a two-handed weapon in place, striking nearby enemies three times at once" |
| dagger flurry | "Channel five rapid dagger strikes over 1.5 seconds" → "Land five rapid dagger strikes at once" |
| arrow volley | "Channel a 3-second arrow volley... ticking 4 times" → "Loose a volley of arrows... striking four times at once" |
| arcane missiles | "Channel four arcane missiles" → "Launch four arcane missiles" |
| Spellstorm | "Channel unstable arcane energy... five rapid ticks" → "Unleash... striking enemies five times in rapid succession" |
| shadow drain | "Channel 5 ticks of shadow drain" → "Tear five pulses of shadow drain from the target" |
| Moonlit Tranquility | "Channel lunar healing over nearby allies" → "Release a wave of lunar healing across nearby allies" |

The heal (Moonlit Tranquility) is phrased to stay accurate **whether it heals instantly or via a HoT**
— no timeframe claim either way. **Zero "channel" claims remain** in any spell description (grep-confirmed).

**The dead `channelDuration`/`channelTime` fields were left in place** — harmless (read nowhere), and
they preserve the aspiration if real channeling is ever built. Removing them would be a separate
cleanup beyond the reword the user asked for.
