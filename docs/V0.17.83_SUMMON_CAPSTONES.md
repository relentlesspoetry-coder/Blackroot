# V0.17.83 — Summon Capstones (all capstones now faithful)

**Type:** Content / gameplay (talent capstones)
**Previous version:** V0.17.82
**New version:** V0.17.83

The V0.17.77 faithful-capstone pass left **two** stand-ins because they needed to *summon* and there was
no proc-callable spawn hook. Now wired — so **all 42 capstones are mechanically faithful**.

---

## What changed (`systems/talent-system.js`, `systems/combat-system.js`, `data/talents.js`)

- **New proc-effect type `summonMinions`** `{ count, petType, color, duration, attackMin, attackMax }` →
  calls the existing `Game.prototype.spawnSummonerTemporarySummons` (the same temp-minion spawner the
  Summoner's own spells use), which creates owned `temporaryPet` entities near the player that fight via
  the normal pet AI.
- **Summoner Planebinder — The Gate Opens**: now actually summons **4 elemental servitors** for 15s on
  cast (plus the +50% summon-damage buff and the target blast).
- **Necromancer Bonecaller — Grave Sovereign**: now actually **raises a skeleton from the slain enemy**
  for 20s (onKill, 25% chance), plus the minion-damage buff.
- **Broadened pet-damage hook** (`damageEntity`): any player-owned pet/minion
  (`source.temporaryPet === true` or `source.owner === player`), not just the single `this.pet`, now
  benefits from `petDamage` talents — so these summons (and all Summoner/Necromancer temp minions) scale
  with the tree.

## Verification (in-browser, port 8899)

Firing a fully-invested Planebinder's capstone spawns **4** `temporaryPet` entities (0 → 4); a
fully-invested Bonecaller's capstone raises a skeleton from a corpse when it procs; `talentPetMultiplier`
applies to a summoned minion; no console errors.

## Talent system status

Every capstone now does exactly what its tooltip says. Remaining talent work is the human-side stuff:
playtest-tuning the pet/tank/healer/support specs (simulation can't judge them), optional painted spec
art (drop-in ready via `assets/talents/<specId>.png`), and a visual sign-off on the themed backgrounds
(screenshots time out on this WebGL page).
