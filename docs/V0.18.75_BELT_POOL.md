# V0.18.75 — Dark Woods Belt Pool (Roadmap Item 4 B)

The itemization expansion begins. The belt slot had exactly **one** item (the Wildpelt Belt); this adds **20 distinct belts** with real profiles, rarities, levels, and sources — every one obtainable and guarded by the boot obtainability audit.

## The 20 belts (`data/items.js`)
`type: armor`, `slot: belt`, levels **1–7**, rarity **8 white / 10 green / 2 blue**. Stats use only real `STAT_KEYS` (no dead stats).

| Profile | Belts |
|---|---|
| Defensive | Frayed Rope, Studded Leather, Ironbark Girdle, Bramblehide Cinch, Ashroot Girdle, **Hollow Warden's Belt** (blue) |
| Offensive | Hunter's Cord, Wolfsinew, Studhide Warbelt, Fanged Warbelt |
| Caster | Woven Reed Sash, Apprentice's Cord, Gloomsilk Sash, **Runethread Belt** (blue) |
| Regeneration | Mendcloth, Lifebloom Girdle |
| Movement/utility | Traveler's, Pathfinder's Cinch |
| Gathering | Forager's Toolbelt |
| **Class-oriented** | **Rootwarden's Girdle** (Warden/Druid/Shaman/Ranger), **Runethread Belt** (caster classes) |

## Sources (every belt obtainable)
- **Vendor (8)** — the basic white belts are sold at the **camp general store** (`systems/event-system.js` `SHOP_ITEM_LISTS`).
- **Loot (10)** — drops from Dark Woods tables (`data/loot-tables.js`): wolves→Wolfsinew, rotlings→Lifebloom, briar boars→Bramblehide, duskwisps→Gloomsilk, hollow stags→Pathfinder's, ashroot horrors→Ashroot, named rares→Fanged + Rootwarden's, and the 2 blue belts (Hollow Warden's + Runethread) from the **common chest** table.
- **Craft (2)** — Ironbark Girdle + Studhide Warbelt at the **campfire** (`data/crafting.js`).
