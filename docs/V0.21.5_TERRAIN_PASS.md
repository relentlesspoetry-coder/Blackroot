# V0.21.5 — The Terrain Graphical Pass

The reference requirement is explicit: *"instead of large flat-color tiles, the ground uses noise,
cracks, stones, dirt variation, moss, and irregular shading."*

## What it draws

`drawChunkGroundTexture`, six layers per ground tile:

| layer | purpose |
|---|---|
| dirt speckle (~0.014/px²) | kills the flat-colour read at close range — without it the eye sees a solid fill however many large rocks sit on top |
| dirt clumps | the mid-frequency band between speckle and the macro wash, which made the ground read as noise-on-flat |
| **pebbles & stones** | three elements each — contact shadow, lit body shaded up from the material base, shaded underside — so they read as objects sitting *on* the earth rather than painted dots. The biggest single contributor. |
| cracks | short jagged runs, not smooth curves; gated on `crackDensity` |
| moss | `mossDensity`, on grass and stone |
| grass blades | short lit strokes on grassy ground |

## Where it draws

Into the **chunk layer**, where terrain is genuinely built — so it is cached and costs nothing per
frame. This is the V0.21.3 finding applied rather than merely recorded: `drawGroundDetail` runs zero
times per frame, so anything installed there would have been invisible.

Fully **deterministic** — every element placed from a per-tile seeded RNG, so a tile always looks
identical and chunk rebuilds are stable.

## Verified

| check | result |
|---|---|
| local surface contrast (mean adjacent-pixel luma delta on the cached layer) | **0.823 → 2.089, +153.8%** |
| repeatability | 2.089 on both arms, identical |
| calls per chunk rebuild | 6,082 |
| boot suite | PASS |

Local contrast is the right metric here: it is what "texture" means numerically, and unlike the
whole-layer standard deviation used in V0.21.3 it is not dominated by material differences.

## Cost, stated plainly

A cold-cache rebuild of the visible chunks measured **~556ms** across three frames. One-time per chunk
and absorbed by the cache during play, but a **visible hitch on zone entry**.

This is the first thing to profile if world loading starts to feel slow. Element counts are a single
set of constants at the top of each block in `drawChunkGroundTexture`.

## Remaining gap to the reference

- saturation 0.379 against 0.444
- fog still drains saturation to 0.017 when weather turns, undoing the V0.21.4 grade
- transition/shoreline tiles and elevation wall faces have no equivalent pass yet
