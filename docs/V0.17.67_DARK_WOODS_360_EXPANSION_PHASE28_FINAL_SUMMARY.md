## What the expansion delivered

Dark Woods is now a large, dense, readable level 1–10 starter wilderness — an old, haunted forest where
dead roads, broken lanterns, spider territory, bandit ruins, wisp magic, ancient roots, and abandoned
human history overlap, with a full zone-exclusive resource identity.

### Zone & structure
- **Dark Woods expanded to 360 x 360 tiles** (`CONFIG.OVERWORLD_MAP_SIZE`; caves/dungeons stay 200x200).
  Save schema bumped to track per-zone sizes; ~7 hardcoded `MAP_SIZE` clamps fixed (movement, render
  window, minimap, click-to-move, item drop, save/load).
- **6 named subregions** with level bands and identity metadata (`data/default-zones.js`
  `dark_woods.regions`, consumed by `getDarkWoodsRegionAt`): Dead Lantern Outskirts (1–3), Bramblefen
  Thicket (3–5), Bandit's Fall (4–7), Stone Hedge Clearing (5–8), Gloamroot Depths (7–10), Silk Web
  Approach (6–10).
- **Main paths + the Dead Lantern Trail** — road spurs into the resized area and a 7-variant lantern
  system (intact/broken/hanging/fallen/wisp/bandit/webbed) auto-assigned by region.
- **Dark Woods now has exactly 2 caves.** Removed Ashroot Hollow / Crystal Grotto / Forgotten Mine /
  Blackroot Catacombs entrances; kept only the two required caves.

### Landmarks
- **Silk Web Cavern is the only dungeon cave** — the canonical 3-floor elite spider dungeon (already
  fully built; audited in Phase 7), with a spider-territory approach (webbed bushes, egg sacs, bones,
  wrapped corpses, broken weapons, dark dirt).
- **Hidden Tree Cave** — the second cave, refined as a small secret exploration cave (concealment ring
  of mossy tall trees / root arches / hanging vines / brush; one hidden lore NPC). A stray second-dungeon
  entrance carved inside it was removed.
- **Bandit's Fall** (Old Ruins / bandit area) — a ruined outpost at (230,60): outer broken wall line,
  south entry gap, central ruined structure, rubble/LOS pockets, camp clutter, and 5 bandit enemy drafts
  (Scout/Thug/Archer/Lookout/named Leader "Rurik the Fallen").
- **Stone Hedge Ruins** — a partial standing-stone ring at (210,250) with a distinct grey-blue mossy
  palette, rune stones, drifting magic residue, and a clear centre.
- **Waypoint added in the Stone Hedge Ruins centre** — the game's first waypoint system (built from
  scratch; `systems/waypoint-system.js`) placed in the exact ring centre, with collision kept clear.
- **Animated waypoint aura** — pulsing glow, ground glow, radial ring, rotating rune ring, and rising
  motes, via the existing effects pipeline (no new render loop); it brightens/widens at night.
- **Wisps placed around Stone Hedge Ruins** — an outer Duskwisp camp and an inner named Lumen-Wisp.

### Visual identity
- **Terrain detail + overlays** — region-depth-based darkening wash and moss coverage that scales with
  danger; fixed a cached-bake ground-detail gap; a terrain-level magical-residue shimmer confined to
  Stone Hedge.
- **Trees / bushes / props** — ancient trees, size-varied dead trees, berry/flowering/mossy bushes,
  moss rocks, natural stumps, and new non-blocking (fallen log, root cluster) and blocking (root
  barrier) props, all scaled by region depth.
- **Micro-landmarks + environmental storytelling** — Abandoned Hunter Camp, Old Forest Well, Collapsed
  Watch Post, Root Shrine; Bandit Ambush Remains, Failed Spider Hunt, Lost Pilgrim Shrine, Old Road
  Collapse (coordinates verified against ~70 mob spawn anchors).
- **Creature territory signposting + ambient life** — webs/arrows/motes/claw-marks/roots that warn
  before each creature family; crows, squirrels, deer, frogs, and bats join the existing fireflies,
  butterflies, moths, beetles (lightweight, camera-culled, cosmetic).
- **Atmosphere / time-of-day** — morning fog pooling in low ground, stronger waypoint glow at night,
  deeper ancient-tree shadows, wisps' night glow, and faint lantern road-glow (all driven by the
  existing day/night light state — no new weather framework).

### Resources (zone-exclusive identity)
- **5 Dark Woods overworld herbs** — Lantern Moss (Gathering 1), Thornberry (2), Gloomcap Mushroom (4),
  Wispbloom (5), Blackroot Herb (7, rare). Placed by a deterministic terrain scan into the correct
  regions; overworld-only.
- **3 Silk Web Cavern herbs** — Webcap Fungus (4), Widow's Veil (6), Queen's Silkroot (8, rare, final
  chamber). The gathering system was made dungeon-aware (floor-specific grid) to reach them.
- **5 Dark Woods overworld fish** — Duskmud Minnow (Fishing 1), Lanternfin (2), Brambleback Pike (4),
  Ghostscale Trout (5), Gloamroot Catfish (7, rare, deep water). Added the missing per-fish level gate.
- **3 Silk Web Cavern cave fish** — Blind Silk Minnow (4), Venomgill Eel (6), Broodpool Angler (8, rare,
  deepest pool). Real water pools were carved into the dungeon floors and fishing enabled there.
- **Resource level / EXP / respawn rules** — Gathering/Fishing level 1–10 EXP tables aligned to the plan
  (100/150/225/325/475/675/950/1300/1750; 5,950 total to level 10); a single authoritative Fishing level;
  first-time-discovery bonuses (+15/30/50/75 by tier); rare-node bonuses (+20/+30); vendor values per the
  plan; and randomized respawn ranges.

### Systems / integrity
- **Persistence + anti-farming** — depletion uses absolute wall-clock timestamps persisted to localStorage
  + the game save, with deterministic node positions, so rare nodes never respawn by leaving/re-entering
  Silk Web Cavern, reloading, or moving off-screen (verified end-to-end).
- **UI messaging + feedback** — "Requires Gathering/Fishing Level X", gather/catch success with total
  "+N EXP", and first-discovery/rare-bonus feedback, reusing the existing panels/log (no HUD redesign).
- **Collision + readability** — audited: all resource nodes, dungeon pools (with fishable shores),
  landmarks, and the waypoint are reachable; boss arenas clear; map edges blocked.
- **Performance** — no per-frame random or duplicate render loops were introduced; `getWorldLightState`
  is memoized per frame.

## Pre-existing bugs found and fixed along the way
- **Phase 16:** `atmosphere-system.js` clamped to the old 200-tile `MAP_SIZE` instead of `activeMapSize()`,
  so fog/fireflies/ambient life silently failed to render across the outer half of the resized zone since
  V0.17.41.
- **Phase 21:** `object-renderer.js` dispatched `silkStrands` to a `drawSilkStrands` that was never
  defined — a render fault every frame in Silk Web Cavern.
- **Phase 23:** `entity-renderer.js` `drawFishingActionOverlay` referenced an undeclared `s` instead of
  its `screen` parameter — a render fault for every fishing player, overworld included.
- **Phase 20:** `tableForResource` fell back to a generic gloomleaf/copper loot table for any node
  without an explicit table, so authored drops (herbs, Mooncap, Darkstone) were mis-yielded.

## Known orphaned content (documented, intact, unreachable)
- `dungeon_glooms_crypt` (was behind the removed Blackroot Catacombs cave) and
  `dungeon_blind_pool_depths` (was behind a removed underwater entrance inside Hidden Tree Cave) — both
  have full data/bosses but no current access point; left in place for a future decision.
