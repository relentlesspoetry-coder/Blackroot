## Display / resolution ownership
Display ownership was found in:

- `core/config.js` for project-wide display metadata

## 1440p implementation

- Added `DISPLAY_TARGET_WIDTH: 2560` and `DISPLAY_TARGET_HEIGHT: 1440` to `DR.CONFIG`.
- Added `Game.prototype.displayTargetSize()` as a small display helper owned by `game.js`.
- Updated `resize()` to preserve responsive CSS sizing while stamping target metadata onto the canvas and root CSS variables.
- Updated viewport metric fallback behavior so headless/zero-size cases use the 2560x1440 target.
- Preserved fullscreen and smaller-screen behavior by keeping actual CSS canvas size tied to the browser viewport.
