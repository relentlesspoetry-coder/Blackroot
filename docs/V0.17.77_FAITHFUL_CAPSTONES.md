# V0.17.77 — Faithful Capstone Mechanics

**Type:** Content / gameplay (talent capstones)
**Previous version:** V0.17.76
**New version:** V0.17.77

The 42 talent capstones shipped as **bounded approximations** — a flat single-target burst or a
temporary buff — because the proc engine only knew a handful of effect types. This pass gives them their
**real spec mechanics**.

---

## New proc-effect vocabulary (`systems/talent-system.js` → `applyTalentProcEffects`)

All built on the existing combat pipeline (`damageEntity` / `applyStatusEffect`), and all run inside the
reentrancy guard + icd, tagged `talentProc:true` so they never recurse into other procs.

| Effect | What it does |
|---|---|
| `damageAoe` | Damage every enemy within `radius` of the target (or the player). |
| `chainStrike` | `count` strikes on random nearby enemies. |
| `detonateDots` | Consume the caster's DoTs on the target for a burst (`pctOfRemaining` of their remaining damage); with `spread`, copy those DoTs to nearby enemies. |
| `applyStatusAoe` | Apply a status (root / freeze / stagger / fear / mesmerize) to nearby enemies. |
| `resetCooldowns` | Zero the cooldowns of named spells for the player. |
| `refreshDots` | Re-up the caster's active DoTs on nearby enemies. |

Plus an `enemiesNear(center, radius, max)` helper over `this.entities`.

## Capstones upgraded (18)

Examples (with matching new descriptions):

- **Necro Plaguebinder — The Plague Spreads**: now actually **spreads every DoT on the corpse to nearby
  enemies** (+ the DoT-damage buff), instead of a flat burst.
- **Shaman Stormcaller — The Storm Answers**: chains **5 lightning strikes** on nearby enemies.
- **Wizard Pyromancer — Conflagration**: **explodes all nearby enemies**; **Frostbinder — Absolute Zero**:
  AoE frost + **freezes** them.
- **Rogue Shadowblade — Death from Shadow**: **re-enters stealth + resets Backstab/Shadowstep**.
- **Rogue Venomfang — The Long Death**: **detonates** the target's poison/bleed.
- **Enchanter Mesmerist — Total Domination**: **mesmerizes all nearby enemies**.
- **Paladin Judicator — Reckoning**: **resets Judgment & Hammer of Reckoning** (+ holy buff).
- AoE / CC upgrades also on: Warden Thornwall, Fighter Sunder the World, Ranger The Snare Closes,
  Assassin The Kill Box, Shaman The Mountain Moves & Ancestral Wrath, Summoner Mass Dismissal, Cleric
  Exorcism, Bard The Last Verse, Enchanter Glyphstorm.

## Verification (in-browser, port 8899)

- Each new effect type in isolation: AoE hit 3/3 nearby enemies; chain dealt damage; root applied 3/3;
  `resetCooldowns` zeroed the named spells and left others; **`detonateDots` + spread** detonated the
  target and copied the DoT to a neighbour.
- End-to-end through `fireTalentProc`: a fully-invested Plaguebinder's `onKill` detonated the corpse and
  **spread its DoT to a nearby enemy**, with the DoT-damage buff active.
- Structural validator: **0 problems** across all 294 nodes; the six new effect types are recognised.
  No console errors.

## Still approximated (need a summon system that doesn't exist)

- **Summoner Planebinder — The Gate Opens** ("summon one of each element") and **Necro Bonecaller — Grave
  Sovereign** ("raise a skeleton from the corpse") keep their pet-damage-buff stand-ins, because there is
  no runtime summon-spawn hook to call from a proc. Flagged for if/when one is added.
