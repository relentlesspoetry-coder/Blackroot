# V0.18.55 — Silk Web Caverns: web cleanup + richer hanging columns

Your screenshot + notes: get rid of the ugly grid/curtain webs; slow the floor spiders; add more crawling spiders to the ceiling webs and the ground web piles; and more hanging columns with bones, crawling spiders, cocooned bodies, and eggs inside.

Showcase: [V0.18.55_rich_columns.png](V0.18.55_rich_columns.png) — thick/medium columns (cocoon + crawling spiders + bones + eggs), the junk pile and bone pile (with crawlers), and the standalone cocoon.

## 1. Removed the ugly geometric webs
The webs you flagged were the old high-fidelity props: **`webCurtain`** (a rectangular net grid — the right structure in your shot), **`heavyWebCurtain`** and **`silkStrands`** (sparse hanging curtains/fans — the left structure). Rendering each confirmed the match.

Rather than editing all ~13 placement sites, I added **one central filter** in `placeSilkObject` (`systems/dungeon-system.js`) that rejects those three types — every placement helper (`scatter`, `add`, `decorateSilkRoomEdges`, `placeSilkDressingObject`) funnels through it, so none of them emit any more. The draw functions are left intact for any other zone.

## 2. Floor spiders slowed
`dungeonSystem.updateSilkCritters` movement speed **1.15 → 0.5** tiles/s. The walk-gait phase scales with speed, so the legs also step slower to match — a natural crawl instead of scurrying.

## 3. More + richer hanging web columns
- **More columns**: `placeSilkCeilingWebsAndJunk` `columnCount` raised (minimal 3→4, big 6→9, pocket 2→3, else 4→6).
- **Richer columns** (`render/object-renderer.js` `drawCeilingWebColumn`), on top of the stacked orb webs:
  - **tiny spiders crawling up + down** the column (animated ping-pong, facing their travel with the leg gait);
  - a **body cocooned in the silk** — new `drawWebCocoon`, a silk-wrapped bundle with wrap bands and a dark face/limb gap (thick columns always, medium sometimes);
  - **bones** tangled at the base, now on **medium** columns too (was thick-only);
  - plus the existing **egg sacs** and the woven **web nest**.
- **Ground piles**: `drawWebJunkPile` and `drawWebbedBonePile` also get **tiny spiders crawling** over them.

No gameplay changes.
